# Hiweb Growth Tools

Suite modular de herramientas interactivas para Hiweb Marketing construida con Next.js, React, TypeScript, Webflow Cloud, Webflow Code Components, SQLite, KV Store y n8n.

## Herramientas

- Quiz multistep de diagnóstico de marketing digital.
- Calendario dinámico para agendar llamadas.
- Calculadora de ROI de marketing digital.

## Desarrollo local

```bash
npm install
npm run dev
